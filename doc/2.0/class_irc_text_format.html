QString IrcTextFormat::toHtml ( const QString &  text) const

Converts text to HTML. This function parses the text and replaces IRC-style formatting like colors, bold and underline to the corresponding HTML formatting. Furthermore, this function detects URLs and replaces them with appropriate HTML hyperlinks.

Note:
URL detection can be disabled by setting an empty urlRegExp.
