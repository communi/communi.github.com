Communi - a cross-platform IRC framework written with Qt

Introduction

Communi is a cross-platform IRC framework written with Qt. IRC (Internet Relay Chat protocol) is a simple text-based communication protocol.

Installation

To build and install Communi, do the standard mantra:

$ qmake
$ make
$ sudo make install

See the INSTALL file for more details.

Usage

Add the following line to your qmake project (.pro) file:

CONFIG += communi

This adds the necessary include paths and linker rules in order to use the library.

Communi in a nutshell:

  • IrcSession manages the connection to an IRC server
  • IrcMessage represents a message received from an IRC server via IrcSession::messageReceived().
  • IrcCommand represents a command sent to an IRC server via IrcSession::sendCommand().
